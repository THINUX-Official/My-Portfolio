About Me

Hi, My name is Thinusha Supun Dilhara.

I am an undergraduate student of IJSE (Institute of Software Engineering). I’m currently learning more about Software Engineering, Web development and Internet Technologies. I like to improve my skills with working on projects. Also, I have knowledge of Web Application design, UI/UX Design & Java FX.
